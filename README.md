# coursework4
# Приложение для автоматизации работы логистической компании

Система управления таможенными декларациями для автоматизации процессов таможенного оформления.

##  Функциональности

### Основные модули
- **Управление декларациями** - создание, редактирование, отслеживание статусов
- **Платежная система** - управление оплатами таможенных сборов
- **Учет транспортных средств** - регистрация и управление транспортными средствами
- **Система пользователей** - ролевая модель (Админ, Клиент, Водитель)
- **Активности** - журнал действий пользователей

### Роли пользователей
- **ADMIN** - полный доступ ко всем функциям
- **CLIENT** - управление своими декларациями и платежами
- **DRIVER** - просмотр связанных транспортных средств

##  Технологии

### Backend
- **Java** - основной язык программирования
- **Spring Boot** - фреймворк
- **Spring Security** - аутентификация и авторизация
- **Spring Data JPA** - работа с базой данных
- **PostgreSQL** - система управления базами данных
- **Maven** - управление зависимостями
